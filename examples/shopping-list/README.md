# Shopping List App - JMiniApp Example

A simple and intuitive shopping list application built with vanilla JavaScript, HTML, and CSS. This application demonstrates core JMiniApp concepts with a practical real-world example.

## Features

- ✅ **Add Items**: Add shopping items with name, price, and quantity
- ✅ **View Items**: See all items with detailed information
- ✅ **Remove Items**: Delete individual items or clear entire list
- ✅ **Mark as Purchased**: Toggle purchase status for each item
- ✅ **Persistent Storage**: Data saved in browser's localStorage
- ✅ **Responsive Design**: Works on desktop and mobile devices
- ✅ **Real-time Statistics**: View total items and total price

## How to Run

### Option 1: Live Server (Recommended)
1. Open the project in VS Code
2. Install the "Live Server" extension
3. Right-click on `index.html` and select "Open with Live Server"

### Option 2: Direct File Access
1. Simply open `index.html` in your web browser
2. The application will load with all functionality
